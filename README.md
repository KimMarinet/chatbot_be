# chatbot 프로젝트 -> BE 파트

### ➡️ 개요 
- chatbot의 BackEnd 서버 기동을 위한 프로그램
- 외부 프로그램(Python)에서 사용자 질문에 대한 예측한 내용을 결과로서 반환한다

### ➡️ 기능 설명
- **환경 변수 설정** : [외부 프로그램의 위치]
- 외부 프로그램(Python)을 실행시켜 예측 결과를 받아온다

### ➡️ 코드 리뷰
- **ChatService** : 외부 프로그램(Python)을 실행시키고 결과 값을 받아옵니다
